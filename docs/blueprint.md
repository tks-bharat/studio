# **App Name**: Global AI Connect

## Core Features:

- Webinar Card Display: Display upcoming webinars as interactive 3D cards with title, date, and theme.
- Card Flip Animation: On hover, flip the card to reveal speaker name and duration using smooth CSS transitions.
- Webinar Detail View: Expand the card on click into a detailed view with webinar title, date, global time zones, duration, overview, and speaker information.
- External Registration: Direct users to an external registration link in a new tab when they click the 'Register Now' button for a selected webinar: https://go.neo4j.com/WBR-EDU-260129-Context-Engineering-APAC_Registration.html .
- Contextual Help Tool: Use a LLM tool to generate bullet points of key topics to be covered, extracted from a block of text. Each entry begins with an explanation about the necessity of a solid contextual understanding to avoid inconsistencies and improve reasoning.
- Countdown Timer: Display a countdown timer for the next upcoming webinar.

## Style Guidelines:

- Primary color: Saturated purple (#9400D3) for a futuristic, high-tech feel.
- Background color: Dark desaturated purple (#1E0536) to enhance the dark mode experience and contrast with vibrant elements.
- Accent color: Cyan (#00FFFF) to complement the primary purple, providing highlights and a gradient effect for a modern aesthetic.
- Body and headline font: 'Inter', a grotesque-style sans-serif.
- Use glassmorphism for cards with soft shadows to create depth and a modern aesthetic.
- Use minimalistic icons to represent different time zones, key topics and to indicate webinar features, enhancing the futuristic feel.
- Incorporate subtle hover glows and smooth transitions for a premium, high-end experience.
